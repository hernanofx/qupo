# QUPO - Backend

Backend en Laravel (API REST) con PostgreSQL. Este directorio contendrá la API y los artefactos del backend.

Pasos iniciales (manual):
1. Instalar PHP 8.1+, Composer y dependencias (extensiones pdo_pgsql, mbstring)
2. Crear proyecto: `composer create-project laravel/laravel .`
3. Configurar `.env` con conexión a PostgreSQL y Redis

Docker: hay un `Dockerfile` y `docker-compose.yml` de ejemplo.
