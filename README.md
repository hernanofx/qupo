# QUPO

QUPO es una plataforma B2B2C para comercios y usuarios finales orientada a reservas y suscripciones (peluquerías, psicólogos, personal trainers, etc.).

Objetivo: MVP mobile-first (PWA) con backend en Laravel + PostgreSQL, frontend en React (Vite), pagos recurrentes (Stripe + pasarelas LatAm opcionales).

Estructura del repositorio (inicial):

- `backend/` – Laravel API
- `frontend/` – React PWA (Vite)
- `infra/` – scripts de despliegue / IaC
- `docs/` – especificaciones, backlog y runbooks
